## About

Clear-Cut-ai is a project that aims to create a cutting machine that can cut through any material with a single cutting blade. The project is built using Python and OpenCV.

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Python 3.x
- OpenCV
- Numpy

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ugwuc/Clear-Cut-ai.git
   ```
2. Install dependencies
   ```sh
   pip install -r requirements.txt
   ```

### Usage

1. Run the main.py file
   ```sh
   python main.py
   ```
